ArchiMate CSV formats

Om GEMMA (ArchiMate) in GGM (UML) weer te geven, is besloten om alleen de UML-objecttypen en -relaties in de GEMMA op te nemen. De metadata worden wel meegenomen om uitwisseling van gegevens mogelijk te maken.
  • CSV-formaat voor elementen is:
    1. ID
    2. Type
    3. Name
    4. Documentation
    5. Specialization (optioneel)
  • CSV-formaat voor relaties is:
    1. ID
    2. Type
    3. Name
    4. Source
    5. Target
    6. Documentation
    7. Specialization (optioneel)
