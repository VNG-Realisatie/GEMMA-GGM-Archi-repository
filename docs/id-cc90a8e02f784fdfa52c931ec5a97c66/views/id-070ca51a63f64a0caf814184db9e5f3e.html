GGM data-objectmodellen (iv3) ()
Via deze indelingsview heb je toegang tot alle GGM data-objectmodellen. De data-objectmodellen bevatten een ArchiMate weergave van het GGM UML informatiemodel. De GGM UML classes, enumeraties en relations zijn ingelezen en omgezet naar ArchiMate objecten. Vervolgens zijn er met een script per iv3-domein views gegenereerd.
