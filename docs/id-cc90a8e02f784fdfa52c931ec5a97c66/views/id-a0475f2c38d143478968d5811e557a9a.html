GEMMA bedrijfsobjectmodellen (iv3) ()
Via deze indelingsview heb je toegang tot GEMMA bedrijfsobjectmodellen. De GEMMA bedrijfsobjectmodellen bevatten een selectie van de GGM objecten. De views zijn met script per iv3-domein gegenereerd. De bedrijfsobjectmodellen zijn onderdeel van de GEMMA en kunnen gaan afwijken van het onderliggende GGM
